# Webpack 사용방법
웹팩 기본에 대해 알아 봅시다.

#### Getting-Started
1. 글로벌로 webpack 설치

  ```
  npm i webpack -g (처음 webpack 설치할때 필수)
  ```


2. 패키지 json 파일 만들기

  ```
  npm init
  ```


3. index.js 및 index.html을 작성하십시오

  ```
  <!-- index.html -->
  <html>
    <head>
      <title>webpack 2 demo</title>
      <script src="https://unpkg.com/lodash@4.16.6"></script>
    </head>
    <body>
      <script src="app/index.js"></script>
    </body>
  </html>
  ```
